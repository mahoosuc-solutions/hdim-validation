# HDIM Validation Site

Marketing and documentation site for the HDIM Validation demo application.

Built with Next.js 16, React 19, and Tailwind CSS 4.

Part of the [HDIM ecosystem](https://github.com/mahoosuc-solutions/hdim):
- Core platform: `hdim`
- Validation demo: `hdim-validation` (this repo)
- Provider toolkit: `hdim-accelerator`

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Image Generation

Marketing images are generated with DALL-E 3:

```bash
npm run generate-images
```
